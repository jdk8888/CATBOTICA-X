# Reference Images & Concept Art

**Purpose**: Centralized reference images and concept art for the CATBOTICA project.

## Folder Structure

- **`brand/`**: Official brand guidelines (e.g. `CATBOTICA_Brand_Guidelines_March2022.pdf`) — fonts, colors, logo rules, visual identity
- **`2d_concepts/`**: 2D concept art, sketches, mood boards, and design iterations
- **`3d_concepts/`**: 3D concept renders, previews, and visualization tests
- **`final_outputs/`**: Final approved reference images used across all silos for consistency

## Usage Across Silos

All silos should reference images from `references/final_outputs/` for:
- Character design consistency (Silo 02)
- Environment and architecture reference (Silo 03, 04)
- Lighting and mood reference (Silo 04, 05)
- Post-production color grading reference (Silo 07)
