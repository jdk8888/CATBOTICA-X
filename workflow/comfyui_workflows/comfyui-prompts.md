## Usage Tips

1. **Combine Elements**: Mix character + location + mood for scene generation
2. **Reference Bible**: Always check `bible/characters.md` and `bible/worldbuilding.md` for accurate details
3. **Maintain Consistency**: Use same descriptive terms across generations
4. **Style Consistency**: Apply retro-futuristic/cyberpunk aesthetic consistently
5. **Color Coordination**: Match color palettes to location types
